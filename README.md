# Portfolio (W.I.P.)

In this repository you'll find out how I created my online portfolio. I decided to make it open source since I stand behind the #buildinpublic. I hope it helps your creativity or that it is interesting to look at.

This is a Next.js instance with Typescript and Tailwind support (+Three.js eventually).

## Installation

### 1. Install dependencies

Install: yarn install

### 2. Lets get it running

Start developing: yarn dev

#### (Keep in mind that this repository is an ungoing project. After the initial set up, it will evolve more and more)
